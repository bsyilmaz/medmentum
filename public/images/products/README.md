# Ürün Fotoğrafları

Bu klasöre ürün fotoğraflarınızı yükleyebilirsiniz.

## Kullanım

Fotoğrafları buraya yükledikten sonra, Products.jsx dosyasındaki `image_url` değerlerini şu şekilde güncelleyin:

```javascript
image_url: '/images/products/your-image-name.jpg'
```

## Önerilen Format

- **Format**: JPG, PNG veya WebP
- **Boyut**: 1200x800px veya daha büyük
- **Dosya Adı**: Küçük harfler ve tire kullanın (örn: `ai-diagnostics.jpg`)

## Örnek Dosya İsimleri

- `ai-diagnostics-platform.jpg`
- `telemedicine-suite.jpg`
- `health-records-system.jpg`
